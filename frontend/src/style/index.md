# Tables

Tables should be styled with default HTML markup

```html_example
<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
            <th>Header 3</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <th>Row header</th>
            <td>Data cell 1</td>
            <td>Another data cell</td>
        </tr>
        <tr>
            <th>Row header 2</th>
            <td>Data cell 2</td>
            <td>Another data cell</td>
        </tr>
        <tr>
            <th>Row header 3</th>
            <td>Data cell 3</td>
            <td>Another data cell</td>
        </tr>

    </tbody>
</table>

```
